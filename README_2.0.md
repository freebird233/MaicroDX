# Maicrodx 2.0 Model Assembly Guide

<br>

1. **Prerequisites**  
   Before starting, please ensure your tablet is an 11‑ to 13‑inch model with a width **not exceeding 220 mm**.  
   <br><br>

2. **Printing Parts**  
   Print all components from the makerworld universal configuration file. Two engraving styles are available:  
   [Web link](https://makerworld.com.cn/zh/models/1561531)  
   <br><br>
   <img width="600" alt="image" src="https://github.com/user-attachments/assets/1ed6f3ef-a9dc-4e42-98a2-ad5a31cce627" />
      <br><br><br>

3. **Hardware Preparation**  
   The model supports up to 13‑inch tablets, with a volume nearly double that of the previous generation—printing time will also be longer. You can start purchasing hardware while printing.  
   <br><br>
   **30 AWG single‑core silver‑plated wire (at least 5 m)** : ¥4.49  
   https://e.tb.cn/h.hBCuZG2iKpiAnD6?tk=XCFL4pLFaQ9  
   <br>
   **Low‑profile red switches (10 pcs)** : ¥11  
   https://mobile.yangkeduo.com/goods2.html?ps=eyPVs6rclm  
   <br>
   **OTG right‑angle adapter (C male to A female)** : ¥2.73  
   https://mobile.yangkeduo.com/goods2.html?ps=BxALLyKNBC  
   <br>
   **USB 9‑key programmable keyboard** : ¥33.37  
   https://e.tb.cn/h.hByhHTJPLsIOH7V?tk=ob3u4ppTyxL

# Button Control

Disassemble the 9‑key keyboard case and take out the PCB board.  
<img src="https://github.com/user-attachments/assets/c978c0e6-fdb4-48b0-9297-3cd38f5da331" width="400" >  
<br> The image above shows the result after desoldering the black switch sockets with a soldering iron; the solder points should be clearly visible.  
*(Alternatively, you can solder the wires directly onto the switch sockets—the effect is the same.)*  
<br><br><br><br>

<img width="500" alt="80c00f655ec31978e7290bde7d4e652" src="https://github.com/user-attachments/assets/68191312-93d4-4be2-b3b4-c0a97311866e" /> <img width="300" alt="80c00f655ec31978e7290bde7d4e652" src="https://github.com/user-attachments/assets/13a27dda-3d21-4a31-9b79-2fc683db114c" />  
Connect the PCB keyboard to a computer, download the software provided by the seller, and flash the key mappings.  
Consider turning off the LED to save power.  
**Configuration software:** https://www.jianguoyun.com/p/DVZ2nxEQ1raWDRiXhOcFIAA  
<br><br><br><br><br>

<img width="480" alt="3711083a9521bbc4f4f6488003fff1e" src="https://github.com/user-attachments/assets/4b009ba8-dc1f-4dcd-bda6-332664c53bbc" /> <img width="310" alt="3711083a9521bbc4f4f6488003fff1e" src="https://github.com/user-attachments/assets/80e9c5f5-3ccb-4673-8051-260165546d74" /><br>  
Insert the low‑profile red switches into the 8 slots on the button wheel. The two pins of each switch should face toward the center of the wheel for proper installation.  
<br><br><br><br><br>

<img width="800" height="1015" alt="image" src="https://github.com/user-attachments/assets/d4dccb56-dccd-4609-9205-d288a339b9d5" />  
<br>
The button control section is now complete. It should look like this in the end.  
After threading the wires through the drilled holes, apply AB adhesive around the hole center to secure them and prevent breakage from external force.  
<br><br><br>

# Other Notes
1. The keyboard base (wheel base) connects to the main body via alignment keys, with a 1 mm height allowance. It is recommended to attach 1 mm thick double‑sided tape to the alignment keys on the main body first.
